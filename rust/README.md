# Rust Book Tiếng Việt (Bản tách chương)

Tài liệu này viết lại nội dung cốt lõi từ **The Rust Programming Language** theo dạng nhiều tệp nhỏ để dễ học, dễ tra cứu và thuận tiện mở rộng.

## Cách dùng bộ tài liệu

- Đọc tuần tự từ chương 01 đến 12 nếu bạn mới bắt đầu.
- Dùng mục “Tra cứu nhanh” để tìm chủ đề cụ thể.
- Mỗi chương có:
  - Tóm tắt kiến thức bắt buộc.
  - Ví dụ code ngắn.
  - Checklist cần nắm.
