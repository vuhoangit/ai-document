# Retrieval-Augmented Generation Encyclopedia

## Purpose
This collection is a standardized knowledge source for building and operating RAG systems. It is designed to:

- Align definitions, terminology, and best practices.
- Improve retrieval quality for technical user questions.
- Support chunking, embedding, and reliable source citation.

## Authoring conventions

1. Use clear section titles focused on one topic.
2. Prefer short paragraphs and lists for chunk-friendly content.
3. Repeat key retrieval terms intentionally.
4. Avoid ambiguity; include applicability conditions and trade-offs.
5. For recommendations, state when to use and when not to use.
