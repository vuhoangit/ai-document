# HTML

Tài liệu nền tảng về HTML (HyperText Markup Language) dùng để cấu trúc nội dung trang web.

## Nội dung gợi ý

- Cấu trúc tài liệu HTML cơ bản
- Semantic HTML (header, nav, main, section, article, footer)
- Form và các input type phổ biến
- Media (image, audio, video, iframe)
- Accessibility cơ bản (aria-label, alt, heading structure)
- SEO on-page với HTML
