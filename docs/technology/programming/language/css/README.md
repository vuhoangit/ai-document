# CSS

Tài liệu nền tảng về CSS (Cascading Style Sheets) dùng để định dạng giao diện web.

## Nội dung gợi ý

- Selectors, specificity và cascade
- Box model, display, position
- Flexbox và Grid layout
- Responsive design (media queries)
- CSS variables và tổ chức style
- Animation và transition cơ bản
